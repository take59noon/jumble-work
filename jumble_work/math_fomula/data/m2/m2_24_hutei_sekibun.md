### 基本式  
$$F'(x) = f(x)$$
$$\int_{}^{}{f(x)dx} = F(x) + C$$
<br>

$$\int_{}^{}{x^{n}dx} = \dfrac{1}{n + 1}x^{n + 1} + C$$
$$\int_{}^{}{(ax + b)^{n}dx} = \dfrac{1}{a(n + 1)}(ax + b)^{n + 1} + C$$
$$\int_{}^{}{\left\{ kf(x) + lg(x) \right\} dx} = k\int_{}^{}{f(x)dx} + l\int_{}^{}{g(x)dx}$$
<br>

### 計算式の例  
$$\int_{}^{}{( - 3)dx} = - 3x + C$$
$$\int_{}^{}{\left( 4x^{3} - 3x^{2} + 1 \right)dx} = 4x^{4} - x^{3} + x + C$$
$$\int_{}^{}{(x + 1)^{3}dx} = \dfrac{1}{4}(x + 1)^{4} + C$$
$$\int_{}^{}{\left( t^{3} + 4t^{2} + 1 \right)dt} = \dfrac{1}{4}t^{4} + \dfrac{4}{3}t^{3} + t + C$$
