### 基本式

$$a^{p} = M\ \  \Longleftrightarrow \ \ p = \log_{a}M$$
<br>

$$\log_{a}a^{p} = p$$
$$\log_{a}a = 1$$
$$\log_{a}1 = 0$$
$$\log_{a}\dfrac{1}{a} = - 1$$
<br>

$$\log_{a}{MN} = \log_{a}M + \log_{a}N$$
$$\log_{a}\dfrac{M}{N} = \log_{a}M - \log_{a}N$$
$$\log_{a}M^{k} = k\log_{a}M$$
$$\log_{a}\dfrac{1}{N} = - \log_{a}N$$
$$\log_{a}\sqrt[n]{M} = \dfrac{1}{n}\log_{a}M$$
<br>

### 底の変換

$$\log_{a}b = \dfrac{\log_{c}b}{\log_{c}a}$$
<br>

### 計算の例

$$\log_{\frac{1}{3}}9 = \log_{\frac{1}{3}}\left( \dfrac{1}{3} \right)^{- 2} = - 2$$
<br>

$$\log_{3}18 - \log_{3}2 = \log_{3}\dfrac{18}{2} = \log_{3}9 = \log_{3}3^{2} = 2$$
<br>

$$\begin{aligned}
2\log_{2}\sqrt{2} - \dfrac{1}{2}\log_{2}3 + \log_{2}\dfrac{\sqrt{3}}{2} 
&= \log_{2}\left( \sqrt{2} \right)^{2} - \log_{2}3^{\frac{1}{2}} + \log_{2}\dfrac{\sqrt{3}}{2} \\[.5em]
&= \log_{2}\left( \dfrac{2}{\sqrt{3}} \cdot \dfrac{\sqrt{3}}{2} \right) \\[.5em]
&= \log_{2}1 = 0
\end{aligned}$$
<br>

$$\log_{4}32 = \dfrac{\log_{2}32}{\log_{2}4} = \dfrac{\log_{2}2^{5}}{\log_{2}2^{2}} = \dfrac{5}{2}$$
