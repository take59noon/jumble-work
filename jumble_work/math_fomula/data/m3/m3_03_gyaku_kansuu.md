### 逆関数
$$f^{- 1}(x)$$
$$b = f(a)\ \  \Longleftrightarrow \ \ a = f^{- 1}(b)$$
<br>

### 合成関数
$$g\left( f(x) \right) = (g \circ f)(x)$$
$$f\left( g(x) \right) = (f \circ g)(x)$$
<br>

$$\left( f \circ f^{- 1} \right)(x) = f\left( f^{- 1}(x) \right) = x$$
$$(f \circ g)^{- 1}(x)$$
$$\left( g^{- 1} \circ f^{- 1} \right)(x) = g^{- 1}\left( f^{- 1}(x) \right)$$
<br>

### 計算式の例

$f(x) = 2x + 3$，$g(x) = \dfrac{x + 1}{x - 3}$ とする．  

$$x = 2y + 3\ \  \Longleftrightarrow \ \ y = \dfrac{x - 3}{2}$$
$$f^{- 1}(x) = \dfrac{x - 3}{2}$$
<br>

$$x = \dfrac{y + 1}{y - 3}\ \  \Longleftrightarrow \ \ x(y - 3) = y + 1\ \  \Longleftrightarrow \ \ y = \dfrac{3x + 1}{x - 1}$$
$$g^{- 1}(x) = \dfrac{3x + 1}{x - 1}$$
<br>

$$\begin{aligned}
(f \circ g)(x) 
&= f\left( g(x) \right) \\[.3em]
&= 2g(x) + 3 = 2 \cdot \dfrac{x + 1}{x - 3} + 3 \\[.8em]
&= \dfrac{5x - 7}{x - 3}
\end{aligned}$$
