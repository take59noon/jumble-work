### 基本式
$$y = \dfrac{k}{x} \quad (k \neq 0)$$
$$y = \dfrac{ax + b}{cx + d} \quad (ad - bc \neq 0,\ \ \ c \neq 0)$$
$$y = \dfrac{k}{x - p} + q$$
<br>

### 漸近線
$$x = p,\enspace y = q$$
<br>

### 計算式の例
$$y = \dfrac{3x - 2}{x - 1} = \dfrac{3(x - 1) + 1}{x - 1} = \dfrac{1}{x - 1} + 3$$
$$y = \dfrac{6x + 5}{3x + 1} = \dfrac{2(3x + 1) + 3}{3x + 1} = \dfrac{3}{3x + 1} + 2 = \dfrac{1}{x + \dfrac{1}{3}} + 2$$
