### 基本式  
$$\int_{}^{}{\left\{ kf(x) + lg(x) \right\} dx} = k\int_{}^{}{f(x)dx} + l\int_{}^{}{g(x)dx}$$
$$\int_{}^{}{x^{\alpha}dx} = \dfrac{1}{\alpha + 1}x^{\alpha + 1} + C$$
<br>

### 重要関数の不定積分  
$$\int_{}^{}\dfrac{dx}{x} = \log|x| + C$$
<br>
$$\int_{}^{}{\sin xdx} = - \cos x + C$$
$$\int_{}^{}{\cos xdx} = \sin x + C$$
<br>
$$\int_{}^{}\dfrac{dx}{\cos^{2}x} = \tan x + C$$
$$\int_{}^{}\dfrac{dx}{\sin^{2}x} = - \dfrac{1}{\tan x} + C$$
<br>
$$\int_{}^{}{e^{x}dx} = e^{x} + C$$
$$\int_{}^{}{a^{x}dx} = \dfrac{a^{x}}{\log a} + C$$
<br>

### 計算式の例

$$\int_{}^{}{\sqrt[4]{x^{3}}dx} = \int_{}^{}{x^{\frac{3}{4}}dx} = \dfrac{4}{7}x^{\frac{7}{4}} + C = \dfrac{4}{7}x\sqrt[4]{x^{3}} + C$$

$$\begin{aligned}
\int_{}^{}{\dfrac{\left( \sqrt{x} - 2 \right)^{3}}{x}dx} 
&= \int_{}^{}{\left( \sqrt{x} - 6 + \dfrac{12}{\sqrt{x}} - \dfrac{8}{x} \right)dx} \\[.8em]
&= \dfrac{2}{3}x^{\frac{3}{2}} - 6x + 24x^{\frac{1}{2}} - 8\log|x| + C \\[.8em]
&= \dfrac{2}{3}x\sqrt{x} - 6x + 6\sqrt{x} - 8\log x + C
\end{aligned}$$
