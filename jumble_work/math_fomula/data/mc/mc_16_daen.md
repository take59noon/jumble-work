### 基本式  

焦点が $x$ 軸上にある楕円  

$$\dfrac{x^{2}}{a^{2}} + \dfrac{y^{2}}{b^{2}} = 1$$
$$(a > b > 0)$$

$$\big( \sqrt{a^{2} - b^{2}},\ 0 \big),\enspace \big( {-\sqrt{a^{2} - b^{2}}},\ 0 \big)$$
<br>

焦点が $y$ 軸上にある楕円  

$$\dfrac{x^{2}}{a^{2}} + \dfrac{y^{2}}{b^{2}} = 1$$
$$(b > a > 0)$$
$$\big( 0,\ \sqrt{b^{2} - a^{2}} \big),\enspace \big( 0,\ {-\sqrt{b^{2} - a^{2}}} \big)$$
<br>

楕円 $\dfrac{x^{2}}{a^{2}} + \dfrac{y^{2}}{b^{2}} = 1$ 上の点 $\left( x_{1},\ y_{1} \right)$ における接線の方程式  

$$\dfrac{x_{1}x}{a^{2}} + \dfrac{y_{1}y}{b^{2}} = 1$$
<br>

### 計算式の例

$$\dfrac{x^{2}}{36} + \dfrac{y^{2}}{16} = 1$$
$$16x^{2} + 25y^{2} = 400$$
$$\dfrac{x^{2}}{25} + \dfrac{y^{2}}{16} = 1$$
