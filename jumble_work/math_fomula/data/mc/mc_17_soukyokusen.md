### 基本式

焦点が $x$ 軸上にある双曲線  

$$\dfrac{x^{2}}{a^{2}} - \dfrac{y^{2}}{b^{2}} = 1$$
$$(a > 0,\ b > 0)$$
<br>

中心は原点，頂点は $2$ 点 $(a,\ 0)$，$( -a,\ 0)$  

焦点 $\big( \sqrt{a^{2} + b^{2}},\ 0 \big)$，$\big( {-\sqrt{a^{2} + b^{2}}},\ 0 \big)$

漸近線  
$$\dfrac{x}{a} - \dfrac{y}{b} = 0,\quad \dfrac{x}{a} + \dfrac{y}{b} = 0$$
<br>

焦点が $y$ 軸上にある双曲線  

$$\dfrac{x^{2}}{a^{2}} - \dfrac{y^{2}}{b^{2}} = - 1$$
$$(a > 0,\ b > 0)$$

中心は原点，頂点は2点$(0,\ \ \ b)$，$(0,\ \  - b)$

焦点 $\big( 0,\ \sqrt{a^{2} + b^{2}} \big)$，$\big( 0,\ {-\sqrt{a^{2} + b^{2}}} \big)$  

漸近線  
$$\dfrac{x}{a} - \dfrac{y}{b} = 0,\quad \dfrac{x}{a} + \dfrac{y}{b} = 0$$
<br>

双曲線 $\dfrac{x^{2}}{a^{2}} - \dfrac{y^{2}}{b^{2}} = 1$ 上の点 $\left( x_{1},\ y_{1} \right)$ における接線の方程式  
$$\dfrac{x_{1}x}{a^{2}} - \dfrac{y_{1}y}{b^{2}} = 1$$
<br>

### 計算式の例

$$\dfrac{x^{2}}{16} - \dfrac{y^{2}}{25} = 1$$
$$9x^{2} - 16y^{2} = 144$$
$$\dfrac{x^{2}}{16} - \dfrac{y^{2}}{9} = 1$$
