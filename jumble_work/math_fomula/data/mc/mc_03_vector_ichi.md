### 基本式  

$\mathrm{A}\big( \overrightarrow{a} \big)$，$\mathrm{B}\big( \overrightarrow{b} \big)$，$\mathrm{C}\big( \overrightarrow{c} \big)$ とする．  

$$\overrightarrow{\rm AB} = \overrightarrow{b} - \overrightarrow{a}$$
<br>

線分 ${\rm AB}$ を $m:n$ に内分または外分する点  

内分点 $\mathrm{P}\big( \overrightarrow{p} \big)$  
$$\overrightarrow{p} = \dfrac{n\overrightarrow{a} + m\overrightarrow{b}}{m + n}$$
<br>

外分点 $\mathrm{Q}\big( \overrightarrow{q} \big)$  
$$\overrightarrow{q} = \dfrac{- n\overrightarrow{a} + m\overrightarrow{b}}{m - n}$$
<br>

中点 $\mathrm{M}\big( \overrightarrow{m} \big)$   
$$\overrightarrow{m} = \dfrac{\overrightarrow{a} + \overrightarrow{b}}{2}$$
<br>

$\triangle{\rm ABC}$ の重心 $\mathrm{G}\big( \overrightarrow{g} \big)$  
$$\overrightarrow{g} = \dfrac{\overrightarrow{a} + \overrightarrow{b} + \overrightarrow{c}}{3}$$
<br>

点 $\mathrm{P}$ が直線 ${\rm AB}$ 上にある $\ \Longleftrightarrow$ $\overrightarrow{\rm AP} = k\overrightarrow{\rm AB}$ となる実数 $k$ がある
<br>

### 計算式の例   

$$2\overrightarrow{\rm AP} + 3\overrightarrow{\rm BP} + \overrightarrow{\rm CP} = \overrightarrow{0}$$
<br>

$$\begin{aligned}
& 2\overrightarrow{\rm AP} + 3\big( \overrightarrow{\rm AP} - \overrightarrow{\rm AB} \big) + \big( \overrightarrow{\rm AP} - \overrightarrow{\rm AC} \big) = \overrightarrow{0} \\[.3em]
& 6\overrightarrow{\rm AP} = 3\overrightarrow{\rm AB} + \overrightarrow{\rm AC} \\[.3em]
& \overrightarrow{\rm AP} = \dfrac{3\overrightarrow{\rm AB} + \overrightarrow{\rm AC}}{6} = \dfrac{2}{3} \times \dfrac{3\overrightarrow{\rm AB} + \overrightarrow{\rm AC}}{4} 
\end{aligned}$$
<br>

${\rm BP}:{\rm PE} = s:(1 - s)$，${\rm CP}:{\rm PD} = t:(1 - t)$ とすると  

$$\overrightarrow{\rm AP} = (1 - s)\overrightarrow{\rm AB} + s\overrightarrow{\rm AE} = (1 - s)\overrightarrow{b} + \dfrac{3}{4}s\overrightarrow{c}$$

$$\overrightarrow{\rm AP} = t\overrightarrow{\rm AD} + (1 - t)\overrightarrow{\rm AC} = \dfrac{1}{3}t\overrightarrow{b} + (1 - t)\overrightarrow{c}$$
