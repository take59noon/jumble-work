### 基本式

$$z = a + bi$$
<br>

$$z = r\left( \cos\theta + i\sin\theta \right)$$
$$r = |z| = \sqrt{a^{2} + b^{2}},\enspace \cos\theta = \dfrac{a}{r},\enspace \sin\theta = \dfrac{b}{r}$$
$$\theta = \arg z$$
$$\arg z = \theta_{0} + 2n\pi$$
<br>

$$z_{1} = r_{1}\left( \cos\theta_{1} + i\sin\theta_{1} \right),\enspace z_{2} = r_{2}\left( \cos\theta_{2} + i\sin\theta_{2} \right)$$
<br>

$$z_{1}z_{2} = r_{1}r_{2}\left\{ \cos\left( \theta_{1} + \theta_{2} \right) + i\sin\left( \theta_{1} + \theta_{2} \right) \right\}$$
$$\left| z_{1}z_{2} \right| = \left| z_{1} \right|\left| z_{2} \right|,\enspace \arg{z_{1}z_{2}} = \arg z_{1} + \arg z_{2}$$
$$\dfrac{z_{1}}{z_{2}} = \dfrac{r_{1}}{r_{2}}\left\{ \cos\left( \theta_{1} - \theta_{2} \right) + i\sin\left( \theta_{1} - \theta_{2} \right) \right\}$$
$$\left| \dfrac{z_{1}}{z_{2}} \right| = \dfrac{\left| z_{1} \right|}{\left| z_{2} \right|},\enspace \arg\dfrac{z_{1}}{z_{2}} = \arg z_{1} - \arg z_{2}$$
<br>

#### ド・モアブルの定理

$$\left( \cos\theta + i\sin\theta \right)^{n} = \cos{n\theta} + i\sin{n\theta}$$
<br>

$1$ の $n$ 乗根  
 
$$z_{k} = \cos\dfrac{2k\pi}{n} + i\sin\dfrac{2k\pi}{n}$$
<br>

### 計算式の例  

$$\begin{aligned}
\big( -\sqrt{3} + i \big)^{- 6} 
&= \left\{ 2\left( - \dfrac{\sqrt{3}}{2} + \dfrac{1}{2}i \right) \right\}^{- 6} = 2^{- 6}\left( \cos\dfrac{5\pi}{6} + i\sin\dfrac{5\pi}{6} \right)^{- 6} \\
&= \dfrac{1}{64}\left\{ \cos( - 5\pi) + i\sin( - 5\pi) \right\}^{- 6} = - \dfrac{1}{64}
\end{aligned}$$
