### 基本式

$\overrightarrow{\rm OA} = \overrightarrow{a}$，$\overrightarrow{\rm OB} = \overrightarrow{b}$，$\overrightarrow{\rm OC} = \overrightarrow{c}$ とする．  

$$\overrightarrow{p} = s\overrightarrow{a} + t\overrightarrow{b} + u\overrightarrow{c}$$
<br>

$$\overrightarrow{a} = \left( a_{1},\ a_{2},\ a_{3} \right),\ \overrightarrow{b} = \left( b_{1},\ b_{2},\ b_{3} \right)$$
$$\big| \overrightarrow{a} \big| = \sqrt{{a_{1}}^{2} + {a_{2}}^{2} + {a_{3}}^{2}}$$
$$k\overrightarrow{a} + l\overrightarrow{b} = k\left( a_{1},\ a_{2},\ a_{3} \right) + l\left( b_{1},\ b_{2},\ b_{3} \right) = \left( ka_{1} + lb_{1},\ ka_{2} + lb_{2},\ ka_{3} + lb_{3} \right)$$
<br>

$$\mathrm{A}\left( a_{1},\ a_{2},\ a_{3} \right),\ \mathrm{B}\left( b_{1},\ b_{2},\ b_{3} \right)$$
$$\overrightarrow{\rm AB} = \left( b_{1} - a_{1},\ b_{2} - a_{2},\ b_{3} - a_{3} \right)$$
$$\big| \overrightarrow{\rm AB} \big| = \sqrt{\left( b_{1} - a_{1} \right)^{2} + \left( b_{2} - a_{2} \right)^{2} + \left( b_{3} - a_{3} \right)^{2}}$$
<br>

$\overrightarrow{\rm AB} = \overrightarrow{b}$，$\overrightarrow{\rm AC} = \overrightarrow{c}$，$\overrightarrow{\rm AD} = \overrightarrow{d}$  

$\overrightarrow{\rm AB} = \overrightarrow{b}$，$\overrightarrow{\rm AD} = \overrightarrow{d}$，$\overrightarrow{\rm AE} = \overrightarrow{e}$  

$\overrightarrow{\rm AC} = \overrightarrow{c}$，$\overrightarrow{\rm AF} = \overrightarrow{f}$，$\overrightarrow{\rm AH} = \overrightarrow{h}$  
<br>

$$\overrightarrow{x} = \overrightarrow{a} + t\overrightarrow{b}$$
$$\big| x\overrightarrow{a} + y\overrightarrow{b} + \overrightarrow{c} \big|$$
