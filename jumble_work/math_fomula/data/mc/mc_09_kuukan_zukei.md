### 基本式  

$2$ 点 $\mathrm{A}\left( x_{1},\ y_{1},\ z_{1} \right)$，$\mathrm{B}\left( x_{2},\ y_{2},\ z_{2} \right)$ を結ぶ線分 ${\rm AB}$ について  

線分 ${\rm AB}$ を $m:n$ に内分する点の座標  
$$\left( \dfrac{nx_{1} + mx_{2}}{m + n},\ \dfrac{ny_{1} + my_{2}}{m + n},\ \dfrac{nz_{1} + mz_{2}}{m + n} \right)$$
<br>

線分 ${\rm AB}$ を $m:n$ に外分する点の座標  
$$\left( \dfrac{- nx_{1} + mx_{2}}{m - n},\ \dfrac{- ny_{1} + my_{2}}{m - n},\ \dfrac{- nz_{1} + mz_{2}}{m - n} \right)$$
<br>

#### 軸に垂直な平面の方程式  

点 $\mathrm{P}\left(a,\ b,\ c \right)$ を通り，$x$ 軸に垂直な平面の方程式は，$x = a$  
点 $\mathrm{P}\left(a,\ b,\ c \right)$ を通り，$y$ 軸に垂直な平面の方程式は，$y = b$  
点 $\mathrm{P}\left(a,\ b,\ c \right)$ を通り，$z$ 軸に垂直な平面の方程式は，$z = c$  
<br>

#### 球面の方程式

$$(x - a)^{2} + (y - b)^{2} + (z - c)^{2} = r^{2}$$
$$x^{2} + y^{2} + z^{2} = r^{2}$$
<br>

#### 球面の方程式の一般形

$$x^{2} + y^{2} + z^{2} + Ax + By + Cz + D = 0$$
$$A^{2} + B^{2} + C^{2} - 4D > 0$$
$$\left( x - \dfrac{A}{2} \right)^{2} + \left( y - \dfrac{B}{2} \right)^{2} + \left( z - \dfrac{C}{2} \right)^{2} = \dfrac{A^{2} + B^{2} + C^{2} - 4D}{4}$$
<br>

中心が点 $\left( \dfrac{A}{2},\ \dfrac{B}{2},\ \dfrac{C}{2} \right)$，半径が $\dfrac{\sqrt{A^{2} + B^{2} + C^{2} - 4D}}{2}$ の球面
