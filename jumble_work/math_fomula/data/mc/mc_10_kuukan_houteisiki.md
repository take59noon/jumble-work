### 基本式

#### 平面の方程式

点 $\mathrm{A}\big( \overrightarrow{a} \big)$ を通り，$\overrightarrow{0}$ でないベクトル $\overrightarrow{n}$ に垂直な平面  

$$\overrightarrow{n} \cdot \big( \overrightarrow{p} - \overrightarrow{a} \big) = 0$$
<br>

$\mathrm{A}( x_{1},\ y_{1},\ z_{1} )$，$\mathrm{P}(x,\ y,\ z)$，$\overrightarrow{n} = (a,\ b,\ c )$ とすると  

$$a( x - x_{1} ) + b( y - y_{1} ) + c( z - z_{1} ) = 0$$
<br>

#### 平面の方程式の一般形

$$ax + by + cz + d = 0$$
<br>

点 $\mathrm{P}\left( x_{1},\ y_{1},\ z_{1} \right)$ と平面 $ax + by + cz + d = 0$ の距離 $h$  

$$h = \dfrac{\big| ax_{1} + by_{1} + cz_{1} + d \big|}{\sqrt{a^{2} + b^{2} + c^{2}}}$$
<br>

#### 直線の方程式

点 $\mathrm{A}\big( \overrightarrow{a} \big)$ を通り，$\overrightarrow{0}$ でないベクトル$\overrightarrow{d}$ に平行な直線  

$$\overrightarrow{p} = \overrightarrow{a} + t\overrightarrow{d}$$
<br>

$\mathrm{A}\left( x_{1},\ y_{1},\ z_{1} \right)$，$\mathrm{P}(x,\ y,\ z)$，$\overrightarrow{d} = \left(l,\ m,\ n \right)$ とすると  

$$x = x_{1} + lt,\ y = y_{1} + mt,\ z = z_{1} + nt$$
$$\dfrac{x - x_{1}}{l} = \dfrac{y - y_{1}}{m} = \dfrac{z - z_{1}}{n}$$
<br>

$2$ 点 $\mathrm{A}\big( \overrightarrow{a} \big)$，$\mathrm{B}\big( \overrightarrow{b} \big)$ を通る直線  

$$\overrightarrow{p} = \overrightarrow{a} + t\big( \overrightarrow{b} - \overrightarrow{a} \big) = (1 - t)\overrightarrow{a} + t\overrightarrow{b}$$
<br>

### 計算式の例

$3$ 点 $\mathrm{A}\left(1,\ 0,\ 0 \right)$，$\mathrm{B}\left(0,\ -2,\ 0 \right)$，$\mathrm{C}\left(0,\ 0,\ 3 \right)$ を通る平面 $\alpha$ の方程式  

$\overrightarrow{n} = \left(a,\ b,\ c \right)$  

$\overrightarrow{n}\bot\overrightarrow{\rm AB}$，$\overrightarrow{n}\bot\overrightarrow{\rm AC}$ より，$\overrightarrow{n} \cdot \overrightarrow{\rm AB} = 0$，$\overrightarrow{n} \cdot \overrightarrow{\rm AC} = 0$  

$\overrightarrow{\rm AB} = \left( -1,\ -2,\ 0 \right)$，$\overrightarrow{\rm AB} = \left( -1,\ 0,\ 3 \right)$ より，$- a - 2b = 0,\ - a + 3c = 0$  
<br>

$a = - 2b$，$a = 3c$ より，$\overrightarrow{n} = \left(6,\ -3,\ 2 \right)$ としてよい．  

$6(x - 1) - 3y + 2z = 0$　すなわち　$6x - 3y + 2z - 6 = 0$
