### 基本式

#### ベクトルの表現  

$$\overrightarrow{\rm AB} = \overrightarrow{b}, \enspace \overrightarrow{\rm AC} = \overrightarrow{c}$$
$$\overrightarrow{\rm AB} = \overrightarrow{a},\enspace \overrightarrow{\rm AF} = \overrightarrow{b}$$
<br>

$$\overrightarrow{a},\enspace \overrightarrow{b},\enspace \overrightarrow{c},\enspace \overrightarrow{d},\enspace \overrightarrow{e}$$
$$\overrightarrow{p},\enspace \overrightarrow{q},\enspace \overrightarrow{u},\enspace \overrightarrow{v},\enspace \overrightarrow{x},\enspace \overrightarrow{y}$$
<br>

#### ベクトルの成分
$$\overrightarrow{a} = \left( a_{1},\enspace a_{2} \right),\enspace \overrightarrow{b} = \left( b_{1},\enspace b_{2} \right)$$
$$\big| \overrightarrow{a} \big| = \sqrt{{a_{1}}^{2} + {a_{2}}^{2}}$$
$$k\overrightarrow{a} + l\overrightarrow{b} = k\left( a_{1},\enspace a_{2} \right) + l\left( b_{1},\enspace b_{2} \right) = \left( ka_{1} + lb_{1},\enspace ka_{2} + lb_{2} \right)$$
<br>

$$\mathrm{A}\left( a_{1},\enspace a_{2} \right),\enspace \mathrm{B}\left( b_{1},\enspace b_{2} \right)$$
$$\overrightarrow{\rm AB} = \left( b_{1} - a_{1},\enspace b_{2} - a_{2} \right)$$
$$\big| \overrightarrow{\rm AB} \big| = \sqrt{\left( b_{1} - a_{1} \right)^{2} + \left( b_{2} - a_{2} \right)^{2}}$$
<br>

#### 平行四辺形であるための条件

$$\overrightarrow{\rm AD} = \overrightarrow{\rm BC}$$

または  

$$\overrightarrow{\rm AB} = \overrightarrow{\rm DC}$$
