### 基本式  

$$a_{n + 1} = a_{n} + d$$
<br>

### 初項 $a$，公差 $d$ の等差数列 $\{ a_{n} \}$ の一般項

$$a_{n} = a + (n - 1)d$$
<br>


### 初項 $a$，公差 $d$，末項 $l$，項数 $n$ の等差数列の和 $S_{n}$

$$S_{n} = \dfrac{1}{2}n(a + l)$$
$$S_{n} = \dfrac{1}{2}n\left\{ 2a + (n - 1)d \right\}$$
<br>

$$1 + 2 + 3 + \cdots + n = \dfrac{1}{2}n(n + 1)$$
$$1 + 3 + 5 + \cdots + (2n - 1) = n^{2}$$
<br>

### 計算式の例  

#### 等差数列をなす $3$ 数 $a - d$，$a$，$a + d$

$$(a - d) + a + (a + d) = 15$$
$$(a - d)a(a + d) = 80$$
$$a = 5 \quad  d^{2} = 9 \quad d = \pm 3$$
<br>

#### 等差数列をなす $3$ 数 $a$，$b$，$c$

$$a + b + c = 15$$
$$abc = 80$$
<br>

$$2b = a + c \quad b = 5$$
$$a + c = 10$$
$$ac = 16$$
$$\begin{aligned}
& t^{2} - 10t + 16 = 0 \\[.3em]
& (t - 2)(t - 8) = 0
\end{aligned}$$
