### 基本式

$$\sum_{k = 1}^{n}c = nc$$
$$\sum_{k = 1}^{n}1 = n$$
<br>

$$\sum_{k = 1}^{n}k = \dfrac{1}{2}n(n + 1)$$
$$\sum_{k = 1}^{n}k^{2} = \dfrac{1}{6}n(n + 1)(2n + 1)$$
$$\sum_{k = 1}^{n}k^{3} = \left\{ \dfrac{1}{2}n(n + 1) \right\}^{2}$$
<br>

$$\sum_{k = 1}^{n}r^{k - 1} = \dfrac{1 - r^{n}}{1 - r} = \dfrac{r^{n} - 1}{r - 1}$$
<br>

$$\sum_{k = 1}^{n}\left( pa_{k} + qb_{k} \right) = p\sum_{k = 1}^{n}a_{k} + q\sum_{k = 1}^{n}b_{k}$$
<br>

#### 階差数列 $\{ b_{k} \}$

$$b_{k} = a_{n + 1} - a_{n} \quad (n \geqq 1)$$
$$a_{n} = a_{1} + \sum_{k = 1}^{n - 1}b_{k} \quad (n \geqq 2)$$
<br>

### 計算式の例

$$\sum_{k = 1}^{n}\left( k^{2} + 2k \right) = \sum_{k = 1}^{n}k^{2} + 2\sum_{k = 1}^{n}k$$
$$\sum_{k = 1}^{n}{2 \cdot 3^{k - 1}} = 2 \cdot \dfrac{3^{n} - 1}{3 - 1} = 3^{n} - 1$$
$$\sum_{k = 1}^{n}(2k - 7) = \dfrac{1}{2}\left\{ - 5 + (2n - 7) \right\} n = n(n - 6)$$
